# Internship Tracker Malaysia

A web application for tracking internship applications with a focus on Malaysian companies.

## Project Description

This is a responsive web application built for IMS566 Advanced Web Design Development and Content Management. The application allows students to track their internship applications to Malaysian companies, monitor application status, and visualize their progress.

## Features Included

- **Authentication System**: Login and registration with hardcoded credentials
- **Dashboard**: Overview of internship applications with charts and statistics
- **My Internships**: Detailed view of all internship applications with filtering capabilities
- **Companies**: Directory of Malaysian companies with filtering by industry
- **Data Visualization**: Charts showing application status and monthly trends
- **Responsive Design**: Works on both desktop and mobile devices

## Frameworks/Libraries Used

- Bootstrap 5 (Frontend framework)
- Chart.js (For doughnut chart on dashboard)
- ApexCharts (For line chart on dashboard)
